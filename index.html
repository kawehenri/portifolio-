Sobre Mim

Sou Kawê Henrique, estudante de Engenharia de Software pela Universidade do Distrito Federal (UDF), com formação técnica em Desenvolvimento de Sistemas pelo SENAI-DF. Atuo como estagiário na DF Informática, onde desenvolvo e mantenho sistemas web utilizando PHP, Laravel, HTML, CSS e JavaScript.

Tenho experiência prática com back-end (Python, PHP, Laravel), front-end (HTML, CSS, JavaScript, React) e banco de dados (MySQL). Gosto de resolver problemas complexos com código limpo e boas práticas de engenharia de software.

Tenho interesse em Inteligência Artificial, Cibersegurança e automação. Estou em constante evolução técnica e aprimorando meu inglês para colaborar em projetos globais.
